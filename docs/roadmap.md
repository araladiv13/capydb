# Vision

CapyDB aims to become an AI-powered engineering platform that helps engineers understand, generate, execute, and troubleshoot database operations through automation, observability, and explainable AI.

The roadmap represents the planned evolution of the platform and is expected to evolve as CapyDB grows.

---

| Phase | Status | Target Version | Focus |
|--------|--------|----------------|-------|
| Phase 0 | ✅ Completed | v0.1 | Foundation |

| Phase 2 | 📅 Planned | v0.3 | Data Layer |
| Phase 3 | 📅 Planned | v0.5 | Enterprise Platform |

---

# Phase 0 — Foundation

## Objective

Build a solid engineering foundation before developing product features.

## Deliverables

- Git Repository
- GitHub Repository
- Docker Compose
- n8n Development Environment
- Engineering Principles
- Git Basics Documentation
- Architecture Documentation
- README
- CHANGELOG

## Outcome

CapyDB has a stable development environment, documentation standards, and engineering workflow.

Status : ✅ Completed

---

# Phase 1 — AI Workflow

## Objective

Build the first AI-powered automation workflow.

## Deliverables

- Webhook Trigger
- OpenAI Integration
- Prompt Engineering
- SQL Generator
- JQL Generator
- Error Handling
- Basic Explainability

## Outcome

Users can submit natural language requests and receive explainable SQL/JQL suggestions.

Status : 🚧 Current

---

# Phase 2 — Data Layer

## Objective

Persist execution data for traceability and future intelligence.

## Deliverables

- PostgreSQL
- Metadata Repository
- Execution History
- Prompt History
- Audit Log
- Workflow History

## Outcome

Every execution can be stored, reviewed, and analyzed.

Status : 📅 Planned

---

# Phase 3 — Enterprise Platform

## Objective

Prepare CapyDB for production environments.

## Deliverables

- Authentication
- Multi-Database Support
- Monitoring
- Housekeeping
- Backup
- Audit Trail
- Configuration Management

## Outcome

CapyDB becomes suitable for enterprise deployment.

Status : 📅 Planned

---

# Phase 4 — AI Engineering Platform

## Objective

Transform CapyDB into an intelligent engineering assistant.

## Deliverables

- Root Cause Analysis
- AI Troubleshooting
- Knowledge Base
- Recommendation Engine
- Workflow Optimization
- Operational Intelligence

## Outcome

CapyDB assists engineers in understanding and resolving operational issues using explainable AI.

Status : 🔮 Vision
